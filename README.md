# Database Admin & Security Tool

## Overview
This repository showcases administrative scripts and application logic focused on Database Security, User Management, and Maintenance. 

## Key Features
- **Access Control:** Implementation of SQL Roles (Cashier vs Manager).
- **Data Auditing:** Log tables to track price changes and sensitive data modifications.
- **Performance:** Database indexing for high-speed retail operations.
- **Data Protection:** Delphi logic for automated backup procedures.

## Professional Impact
Essential for retail environments where data integrity and security are top priorities. Demonstrates a deep understanding of the Database Administrator (DBA) role.
